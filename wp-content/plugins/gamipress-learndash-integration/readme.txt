Connect GamiPress with LearnDash

== Description ==

Gamify your [LearnDash](http://www.learndash.com "LearnDash") LMS site thanks to the powerful gamification plugin, [GamiPress](https://wordpress.org/plugins/gamipress/ "GamiPress")!

This plugin automatically connects GamiPress with LearnDash adding new activity events.

= New Events =

= For Quizzes =

* Complete any quiz: When an user finish a quiz
* Complete a specific quiz: When an user finish a specific quiz
* Complete any quiz of a specific course: When an user finish a quiz of a specific course
* Complete any quiz with a minimum percent grade: When an user finish a quiz with a minimum percent grade
* Complete a specific quiz with a minimum percent grade: When an user finish a specific quiz with a minimum percent grade
* Complete a quiz of a specific course with a minimum percent grade: When an user finish a quiz of a specific course with a minimum percent grade
* Complete any quiz with a maximum percent grade: When an user finish a quiz with a maximum percent grade
* Complete a specific quiz with a maximum percent grade: When an user finish a specific quiz with a maximum percent grade
* Complete a quiz of a specific course with a maximum percent grade: When an user finish a quiz of a specific course with a maximum percent grade
* Complete any quiz on a range of percent grade: When an user finish a quiz on a range of percent grade
* Complete a specific quiz on a range of percent grade: When an user finish a specific quiz on a range of percent grade
* Complete a quiz of a specific course on a range of percent grade: When an user finish a quiz of a specific course on a range of percent grade
* Pass any quiz: When an user successfully pass a quiz
* Pass a specific quiz: When an user successfully pass a specific quiz
* Pass a quiz of a specific course: When an user successfully pass a quiz of a specific course
* Fail any quiz: When an user fails a quiz
* Fail a specific quiz: When an user fails a specific quiz
* Fail a quiz of a specific course: When an user fails a quiz of a specific course

= For Topics =

* Complete any topic: When an user finish a topic
* Complete a specific topic: When an user finish a specific topic
* Complete any topic of a specific course: When an user finish a topic of a specific course

= For Assignments =

* Upload an assignment: When an user uploads an assignment
* Upload an assignment to a specific lesson: When an user uploads an assignment to a specific lesson
* Upload an assignment to a specific course: When an user uploads an assignment to a specific course
* Approve an assignment: When an user gets approved an assignment
* Approve an assignment of a specific lesson: When an user gets approved an assignment of a specific lesson
* Approve an assignment of a specific course: When an user gets approved an assignment of a specific course

= For Lessons =

* Complete any lesson: When an user complete a lesson
* Complete a specific lesson: When an user complete a specific lesson
* Complete any lesson of a specific course: When an user complete a lesson of a specific course

= For Courses =

* Complete any course: When an user finish a course
* Complete a specific course: When an user finish a specific course

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
